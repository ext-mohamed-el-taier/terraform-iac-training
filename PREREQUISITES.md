# Prerequisites for Terraform Training

## Labs
- Terraform cli >= 0.14.8
  - ability to download providers from the internet, including (but not limited to):
    - azurerm provider >= 2.41.0
- Azure cli >= 2.17.0 and < 2.30.0
- Bash >= 3 or Powershell >= 5
- git >= 2.30.0
  - ability to clone remote Azure DevOps repos to local filesystem via HTTPS or SSH
- Azure Subscription with access via portal (web browser) and azure cli (`az login`)
- Text editor (e.g. vim) or IDE (e.g. VSCode)
  - optional plugins for syntax completion/highlighting are helpful

## Presentation
- WebEx
- Web browser with internet access
- Access to Azure DevOps Project where training labs are stored
